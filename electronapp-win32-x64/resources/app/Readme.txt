Honeywell Employee Tracker

Please read this file before running the executables:
1. Executable files for Windows and Linux operating systems are given (for x64 architecture).
2. Server is written using the express module of node.js. Hence, Node.js and npm has to be installed to use the server.
4. The express server needs to be established along with the database (given as a dump file), and this can be done 
   by installing the node modules (by executing the command 'npm install')
5. Then the server is switched on by the command npm.
